Hello App PWA

사용 방법
1. 이 폴더를 웹에 올립니다.
2. 아이폰 Safari로 주소를 엽니다.
3. 공유 버튼 → '홈 화면에 추가'를 누릅니다.

중요
- iPhone의 '홈 화면에 추가'는 보통 Safari에서 웹 주소(URL)로 열어야 정상적으로 동작합니다.
- 파일 앱에서 직접 여는 방식은 아이콘, 오프라인, 앱 모드가 제한될 수 있습니다.
